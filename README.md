This repository contains "from scratch" implementations of essential computer science data structures and algorithms using the **C programming language**. 

The project demonstrates low-level memory management (dynamic allocation), pointer manipulation, and algorithmic logic. Each module is self-contained, consisting of a source code file (`.c`) and a corresponding input dataset (`.txt`).

## üìÇ Project Structure

The files are named according to the specific data structures they implement:

| Source Code (`.c`) | Input Data (`.txt`) | Concepts Implemented |
| :--- | :--- | :--- |
| **`SinglyLinkedList_DynamicArray.c`** | `SinglyLinkedList_DynamicArray.txt` | Singly Linked Lists, Dynamic Arrays (`malloc`/`realloc`), File parsing. |
| **`DoublyLinkedList_CircularList.c`** | `DoublyLinkedList_CircularList.txt` | Doubly Linked Lists, Circular Lists, List conversions. |
| **`HashTable_Chaining_Stack.c`** | `HashTable_Chaining_Stack.txt` | Hash Tables (Collision resolution via Chaining), Stacks. |
| **`BinarySearchTree_MaxHeap.c`** | `BinarySearchTree_MaxHeap.txt` | Binary Search Trees (BST), Priority Queues (MaxHeap), Tree Traversals. |
| **`Graph_AdjacencyList_AVLTree.c`** | `Graph_AdjacencyList_AVLTree.txt` | Graphs (Adjacency Lists), Balanced Trees (AVL), Breadth-First Search (BFS). |
